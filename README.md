# GenLayer Mini AI Quest

A beginner-friendly custom GenLayer contract built in GenLayer Studio.

## Features
- Deployable on GenLayer Studio
- Stores a simple question
- Accepts user answers
- Updates score and feedback

## Contract File
- `mini_quest.py`

## Example
Input:
`GenLayer is AI and blockchain`

Output:
- Score = 1
- Feedback = Good answer!

## Purpose
This is my first custom GenLayer builder project.
